### <center>BFS模板</center>

* BFS一般选用队列实现

```java
void BFS() {
	定义队列；
    定义备忘录，记录已经访问的位置
        
	判断边界条件，是否能直接返回结果

    将起始位置加入队列，同时更新备忘录
        
    while(队列不空) {
        获取队列元素个数
        for(元素个数) {
            取出一个位置的节点
            判断是否到达最终位置
            获取它对应的下一个所有的节点
            条件判断，过滤掉不符合条件的位置
            新位置重新加入队列
        }
	}
}
```
